stackoverflowerizer
-------------------

This is a simple Chrome/Safari extension, that redirects to stackoverflow from pages that just copy content.
Currently supporting:

* efreedom
* questionhub
* answerspice
* developerit
* weask
* questionhub
* comanswer
* tuts9
